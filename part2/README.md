### <ins> phonebook </ins>
- Save and display names and numbers.
Possibility to add, update and delete numbers.
- This version reads numbers from `db.json`.
- `baseUrl` changed for next part from `http://localhost:3001/persons` to `/api/persons`

  #### How to run
  - ```npm install```
  - terminal 1: ```npm run dev```
  - terminal 2: ```npm run server```
  - open `http://localhost:5173` in browser
